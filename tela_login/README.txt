#Sobre:

- O projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) de finalização do nosso curso de Desenvolvimento de Sistemas pela escola ETEC (Etec Bento Carlos Botelho do Amaral), Escola Técnica Estadual.

- O objetivo do projeto é criar um site flexível de gestão destinado a diferentes tipos de comércios, desde pequenos negócios até empresas maiores. O principal objetivo é ser um sistema modular, atendendo às expectativas e funcionalidades da empresa e permitindo a personalização conforme os requisitos de cada cliente.

# Principais Funcionalidades:

- O sistema conta com duas interfaces distintas para consumidores e administradores, priorizando a segurança e melhorando a experiência do usuário.

    Interface de Funcionários: Disponibilizada pela empresa, permite que funcionários selecionem seu cargo ao entrar no sistema.

    Interface de Usuários: Não requer escolha de cargo, proporcionando uma experiência simplificada para os consumidores.

- Automação de Processos Financeiros: Inclui cálculos de lucro, impostos e recomposição de estoque, permitindo a modelagem dos dados conforme a necessidade do administrador.

- Gráficos e Previsões: Fornece gráficos para análise detalhada das vendas e cálculos básicos de previsão de lucros por produto ou pela soma de todos os produtos.

- Gestão de Funcionários: Inclui controle de frequência, salários e atestados, além de um sistema de comentários entre gerente e funcionários para garantir a transparência.

- Banco de Dados Independente: Cada unidade de comércio terá um banco de dados independente, enquanto informações essenciais são centralizadas para fornecer aos gestores uma visão ampla do negócio.

- Sistema de Login e Níveis de Acesso: Cada usuário recebe um ID, senha e permissões específicas, garantindo que apenas pessoas autorizadas acessem determinadas informações.

- O sistema de login será inserido o ID, senha, cargo, garantindo que apenas pessoas autorizadas acessem determinadas informações.

- Comentários e Registro de Problemas: O sistema permite o registro de comentários dos gerentes e de problemas, onde os funcionários podem visualizar seu desempenho e contestar comentários se necessário.
